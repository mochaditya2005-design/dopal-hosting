Syarat & Ketentuan Layanan

Sebelum menggunakan Dopal Hosting, harap menyetujui perjanjian berikut.

1. Penggunaan yang Diperbolehkan

Layanan kami dirancang terutama untuk developer, project tugas kuliah mahasiswa, dan profil bisnis standar. Server harus dipergunakan sesuai kaidah hukum yang berlaku di wilayah hukum server beroperasi (dan Republik Indonesia).

2. Konten Terlarang (Zero-Tolerance)

Akun akan ditangguhkan seketika tanpa pengembalian dana apabila pelanggan ditemukan menghosting situs web dengan kategori berikut:

  • Materi Perjudian, Pornografi, eksploitasi, dan pelanggaran SARA.
  • Situs Phising (Penipuan kredensial) atau Distribusi Malware.
  • Pelanggaran Hak Cipta (Nulled Software, Movie Streaming Bajakan, MP3 Ilegal).
  • Website pengalihan traffic yang membebani kinerja tinggi secara kontinyu.

3. Kewajiban & Keamanan Backup

Pelanggan secara eksklusif bertanggungjawab penuh terhadap pencadangan (backup) data personal mereka secara berkala di luar server Dopal Hosting. Kendati kami mungkin melakukan backup mingguan pada sistem root, prosedur tersebut bukan sebuah jaminan dan Dopal tidak bertanggungjawab jika terjadi kerugian insidental berupa kehilangan/korupsi berkas.

4. Aturan Permintaan Instalasi Khusus (Bantuan Deploy)

Fitur penawaran bantuan setup gratis (Deploy Laravel & PHP) terbatas pada instalasi awal dan pengecekan koneksi database. Tim Dopal Hosting tidak wajib menulis, memodifikasi alur logic/sintaks kode (recode), atau memecahkan bug pada script/aplikasi rakitan buatan pelanggan sendiri di luar isu dasar web hosting.

5. Siklus Pembayaran

Invoice layanan berbasis sewa berlangganan (Contoh: Kuota Per Bulan). Fasilitas web hosting dapat mati seketika pada hari di mana jatuh tempo masa aktif berakhir tanpa pembayaran yang dikirim klien (atau dibatalkan sepihak). Bukti pembayaran wajib dirujuk ke Admin CS via WhatsApp.

6. Revisi Ketentuan

Dopal Hosting berhak menambah, menghapus, atau memodifikasi setiap ketentuan di halaman syarat dan ketetapan ini sewaktu-waktu tanpa keharusan mengirim peringatan pemberitahuan sebelumnya, penggunaan lanjutan menandakan kesepakatan.
